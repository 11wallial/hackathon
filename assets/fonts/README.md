# Fonts

Self-hosted so the app loads with no CDN dependency and works offline on first
visit. Only the **latin** subset of each face is vendored to keep the files tiny.

| File | Family | Weight / style | Source |
| --- | --- | --- | --- |
| `poppins-400.woff2` | Poppins | 400 | Google Fonts |
| `poppins-500.woff2` | Poppins | 500 | Google Fonts |
| `lora-400.woff2` | Lora | 400 | Google Fonts |
| `lora-500.woff2` | Lora | 500 | Google Fonts (variable subset, shares the 400 file) |
| `lora-400-italic.woff2` | Lora | 400 italic | Google Fonts |
